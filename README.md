# Electric Vehicle (EV) Market Analysis Dashboard

A clean, dark-themed Power BI data analytics dashboard monitoring global electric vehicle adoption, technological changes, and environmental regulatory alignments. This project separates clean energy vehicle types and manufacturer dominance patterns.

## 📊 Key Metrics & Insights
* **Total Vehicles Tracked:** 150.48K
* **Average Electric Range:** 67.88 miles
* **BEV (Battery Electric Vehicles) Integration:** 117K units (78% of total fleet)
* **PHEV (Plug-in Hybrid Electric Vehicles) Integration:** 34K units (22% of total fleet)

## 📈 Features & Visualizations
* **Historical Adoption S-Curve:** A timeline chart showcasing the massive growth trend in EV models year-over-year, peaking sharply in recent years.
* **Market Share by Manufacturer (Make):** Horizontal distribution tracking market leaders, confirming Tesla's extreme volume dominance (69K vehicles).
* **Geographic Distribution:** A high-contrast map plotting EV ownership concentration across regions.
* **CAFV Eligibility Analysis:** Donut chart tracking Clean Alternative Fuel Vehicle status partitions (showing 46.32% meeting clear eligibility metrics).
* **Product Line Tree Map:** Hierarchical tree map categorizing top EV models, highlighting the extreme adoption rates of Model Y (29K) and Model 3 (28K).

## 🛠️ Tools & Technologies
* **Power BI Desktop:** Tree Mapping, Conditional formatting themes, Advanced DAX, and Data Modeling.

## 📂 How to Use
1. Download the `.pbix` file.
2. Open with **Power BI Desktop**.
3. Use the top filter header panel to drill down into specific cities or segment by BEV vs PHEV vehicle categories.
